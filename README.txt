'오늘은' 설치 안내
1. TodayIsSetup_v1.n.exe를 '관리자 권한'으로 실행
2. 관리자 권한 승인 (UAC 프롬프트)
3. 설치 완료 후 자동 실행
- 사용설명은 프로그램 내부에 있음
- 설정 저장 위치: C:\Users\<사용자>\AppData\Roaming\오늘은\config.json
4. '윈도우 시작 시 자동 실행' 체크하시려면 프로그램 실행파일을 '관리자 권한'으로 실행하셔야 합니다. 체크해제도 마찬가지 입니다.

설치나 사용 시 보안관련메시지가 생길 수 있으나, '허용'해주시면 프로그램 사용시 문제 없습니다. 
이 메시지는 개인개발프로그램이라서 Windows Defender에서 방어정책으로 발생하는 메시지이니 걱정하지 않으셔도 됩니다.
